**QueryTalk: AI-Powered SQL Business Intelligence**
Bridging the gap between natural language and structured data for non-technical stakeholders.

**Product Vision**
In many organizations, non-technical teams (Marketing, Sales, HR) face high latency when requesting data insights from engineering. QueryTalk democratizes data access by allowing users to query SQL databases using plain English. It leverages Gemini 1.5 Flash and LangChain to translate unstructured questions into executable, safe, and accurate SQL.

Key FeaturesNL-to-SQL Translation: Converts complex English questions into syntactically correct SQLite queries using a Retrieval-Augmented approach.  Intelligent SQL Parsing: Implemented custom regex-based cleaning logic to strip LLM conversational "chatter," ensuring 100% execution success for generated queries.Security Guardrails: Integrated a validation layer to intercept and block destructive commands (e.g., DROP, DELETE, UPDATE), protecting data integrity.Trust-Building "Logic Explainer": Features an expandable technical deep-dive that shows the generated SQL and the AI's retrieval strategy for transparency.Automated Data Rendering: Seamlessly transitions from raw SQL results to interactive Pandas dataframes or high-level metrics depending on the query type.  🛠️ Tech StackAI Engine: Google Gemini 1.5 Flash (via langchain-google-genai)   Orchestration: LangChain (SQLDatabaseChain)   Frontend: Streamlit   Backend/DB: Python, SQLAlchemy, SQLite   🏗️ Architecture & WorkflowInput: User asks a question in plain English.Context Injection: The system provides the database schema (metadata) to the LLM.Reasoning: The AI generates the most efficient SQL query to answer the question.Validation: The query is passed through a security filter and a regex cleaner.Execution: The cleaned query runs against the SQLite database.Delivery: Results are presented as a human-readable summary and a data table.📈 Impact for Product ManagementEfficiency: Reduces the dependency on Data Analysts for routine reporting.User Centricity: Solves the "technical barrier" for stakeholders, improving product usability.
